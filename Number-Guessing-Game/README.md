# Number Guessing Game

A simple command-line Python game where the player attempts to guess a randomly generated number between **1 and 9**. The game provides feedback after each guess and continues until the correct number is found.

## Features

* Random number generation using Python's built-in `random` module
* Instant feedback for high and low guesses
* Attempt tracking
* Lightweight and beginner-friendly implementation

## Project Structure

Number-Guessing-Game/

## Requirements

* Python 3.x

## Running the Project

Clone the repository and execute the script:

```bash
python main.py
```

## Example

```text
Number guessing game
Guess a number (between 1 and 9):

5
Your guess was too high: Guess a number lower than 5

2
Your guess was too low: Guess a number higher than 2

3
CONGRATULATIONS! YOU HAVE GUESSED THE NUMBER 3 IN 2 ATTEMPTS!
```
